# My Generative AI Learning Journey

## What is AI?
Artificial Intelligence means making computers act smart like humans.

## What is Machine Learning?
Machine Learning is when computers learn from data instead of being programmed with rules.

## What is Deep Learning?
Deep Learning is a type of Machine Learning that uses neural networks.

## What is a Neural Network?
A Neural Network is a system inspired by the human brain that learns patterns from data.

## What are Transformers?
Transformers are a special type of neural network that understand relationships between words. Models like ChatGPT use transformers.
